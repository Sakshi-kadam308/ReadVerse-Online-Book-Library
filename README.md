# 📚 ReadVerse – Online Book Management System

ReadVerse is a web-based Book Management System developed using JSP, Servlets, MySQL, HTML, CSS, Bootstrap, and JavaScript. The project allows users to browse, search, and manage books through a simple and responsive interface.

## ✨ Features

* User Registration & Login
* Browse Available Books
* Search Books
* Read Book Details
* Responsive User Interface
* Admin Book Management

## 🛠️ Technologies Used

* Java (JSP & Servlets)
* HTML5
* CSS3
* Bootstrap
* JavaScript
* MySQL
* Apache Tomcat
* Eclipse IDE

## 📂 Project Structure

```
src/
WebContent/
WEB-INF/
database/
```

## 🚀 How to Run

1. Clone the repository
2. Import the project into Eclipse
3. Configure Apache Tomcat
4. Create the MySQL database and import the SQL file
5. Update database credentials in the project
6. Run the project on the server
